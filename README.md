# Rgm-Canteen-Food-Order
**Repo**: https://github.com/maheshch6673/Rgm-Canteen-Food-Order
College canteen app for students (React Native)

Internal Android app for Rgm Engineering College students only.

## Features
- Browse canteen menu items
- Add to cart and place orders  
- UPI payments via PhonePe (integration in progress)
